# Next.js Dynamic Routing

## Description
This is a Next.js application that implements dynamic routing to display user profiles based on dynamic data.

## What's Built
- Dynamic routing for user profiles
- Fetching user data from a mock API

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/nextjs-dynamic-routing.git
   cd nextjs-dynamic-routing
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and go to `http://localhost:3000`

## API Docs
- **GET /api/users**: Returns a list of users.
- **GET /api/users/[id]**: Returns a specific user profile.

## Environment Variables
No environment variables are required for this app.
